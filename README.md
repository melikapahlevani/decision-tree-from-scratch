# Decision Tree From Scratch

A Decision Tree implementation developed for an Artificial Intelligence course project.

## Features

* Custom Decision Tree implementation
* Support for different split criteria
* Hyperparameter tuning using Grid Search
* Data preprocessing pipeline
* Missing value handling
* Outlier treatment
* Categorical feature encoding
* Tree visualization

## Technologies

* Python
* Pandas
* NumPy

## Project Overview

This project implements a Decision Tree classifier from scratch and includes data preprocessing, feature selection, hyperparameter optimization, and evaluation.

The preprocessing pipeline includes:

* Removing unnecessary columns
* Handling missing values
* Treating outliers
* Encoding categorical features
* Feature discretization

Grid Search was used to find suitable hyperparameters and improve model performance while reducing overfitting.

## Results

After improving preprocessing and parameter tuning, the model achieved approximately 83% accuracy on the test dataset.
